# OTTO The Builder

OTTO brings zero-pollution dynamic execution to the Sovereign Kernel via the `ottos_outpost` tool. 

## Capabilities
- **Zero-Pollution Synthesis**: Dynamically creating Docker containers filled with exact script dependencies, executing, and tearing down immediately.
- **Native Outpost**: The fallback mode for interacting directly with host-level files when Docker cannot reach them.

By invoking OTTO, users ensure complex environment requirements are handled natively and seamlessly by the Builder Agent.
